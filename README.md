# Bella Curvas - Moda Feminina Plus Size

Um site moderno e impactante para loja de moda feminina plus size, desenvolvido com React e Tailwind CSS.

## 🚀 Início Rápido

```bash
# Instalar dependências
npm install

# Executar em desenvolvimento
npm run dev

# Build para produção
npm run build
```

## 📋 Funcionalidades

- ✅ **Catálogo Completo**: 12 produtos com imagens reais
- ✅ **Carrinho de Compras**: Sistema funcional completo
- ✅ **Filtros Avançados**: Por categoria, tamanho, preço e cor
- ✅ **Design Responsivo**: Mobile, tablet e desktop
- ✅ **Provador Virtual**: Interface preparada para IA
- ✅ **Páginas Completas**: Home, catálogo, sobre, contato, etc.

## 🛠️ Tecnologias

- **React 18** - Framework principal
- **Tailwind CSS** - Estilização responsiva
- **React Router** - Navegação
- **Lucide React** - Ícones
- **Vite** - Build tool

## 📁 Estrutura do Projeto

```
src/
├── components/     # Componentes reutilizáveis
├── pages/         # Páginas da aplicação
├── context/       # Contextos React (carrinho)
├── data/          # Base de dados dos produtos
└── assets/        # Imagens e recursos
```
